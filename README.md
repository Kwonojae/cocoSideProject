# CoCo 개인 프로젝트

좋아하는 디저트 카페를 찾고 유저들과 공유하며 원하는 디저트 카페를 빠르게 찾아보세요

Ceo유저는 카페를 홍보하며 자신의 카페에 있는 디저트를 등록해보세요 !
